# 实验三：静态成员与友元

## 实验内容

### 一、实验要求

#### 实验 1. 友元函数与成绩等级评定

设计一个 Student 类存储学生的私有成绩数据，要求通过友元函数实现成绩查询与等级评定，验证友元突破类封装的特性。

**定义 Student 类**

私有成员：
- 学号 id（int）
- 数学成绩 math（double）
- 语文成绩 chinese（double）

公有成员：
- 构造函数，用于初始化学号和成绩

**定义友元函数 judgeLevel(Student s)**

功能说明：

1. 判断成绩合法性：若 math 或 chinese 为负数 → 输出 "成绩数据非法！"
2. 如果合法，通过对象直接访问私有成员，计算平均分
3. 判断平均成绩等级：
   - ≥90 → A
   - 80~89 → B
   - 60~79 → C
   - <60 → D

**UML 图**

```
Student
────────────────────────────────────────────
-id: int                           // 学号
-math: double                      // 数学成绩
-chinese: double                   // 语文成绩
────────────────────────────────────────────
+Student(id:int, m:double, c:double)  // 构造函数
────────────────────────────────────────────
judgeLevel(s: Student): char       // 友元函数
```

**编程要求**

- main 函数中创建至少 3 个学生对象
- 调用友元函数验证功能

#### 实验 2. 静态成员与库存管理

设计一个 Book 类，利用静态数据成员统计 Book 类创建图书对象的总库存，利用静态成员函数获取库存信息，验证静态成员属于类而非对象的特性。

**定义 Book 类**

普通私有数据成员：
- bool 型变量 flag：标记当前对象图书是否在馆（已借出为 false，未借出或借出已归还为 true）
- 整型的图书编号 bookId
- 字符数组型的图书名称 bookName

静态私有数据成员：
- totalStock（总库存数，初始值为 0）
- borrowedNum（已借出数量，初始值为 0）

**定义成员函数**

| 函数名 | 功能说明 |
|--------|--------|
| 构造函数 | 传入图书编号和名称，每创建 1 本图书对象，totalStock 加 1 |
| borrowBook() | 若 flag 为 1，模拟借书，borrowedNum 加 1，并设置 flag 为 0；若 flag 为 0，输出"本书已借出！" |
| returnBook() | 若 flag 为 1，输出"无待还图书！"；若 flag 为 0，模拟还书，borrowedNum 减 1，并设置 flag 为 1 |
| getTotalStock()（静态） | 返回当前总库存数（totalStock） |
| getLeftStock()（静态） | 计算并返回剩余库存（totalStock - borrowedNum） |

> **注意：** 静态数据成员必须在类外初始化。

**UML 图**

```
Book
────────────────────────────────────────────
-bookId: int                    // 单本图书的唯一编号
-bookName: char[50]             // 书名
-flag: bool = true              // 单本状态（true=在馆，false=借出）
-totalStock: int = 0            // 静态：总库存（下划线表示静态）
-borrowedNum: int = 0           // 静态：已借出数量
────────────────────────────────────────────
+Book(id:int, name:char[])           // 构造函数
+borrowBook(): void                  // 借书操作
+returnBook(): void                  // 还书操作
<<static>>+getTotalStock(): int      // 获取总库存
<<static>>+getLeftStock(): int       // 获取剩余库存
```

**编程要求**

1. 在 main 函数中创建至少 5 个 Book 对象
   - 例如：C++程序设计、数据结构、算法导论、操作系统、计算机网络等

2. 模拟借书、还书操作

3. 通过 `类名::静态函数名` 的方式调用静态函数，输出总库存和剩余库存
