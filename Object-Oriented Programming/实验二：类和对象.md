# 实验二：类和对象

## 实验内容

### 一、实验要求

#### 1. 一元二次方程的求解

一元二次型方程的求解是数学中的常见问题，其中一种求解方法是"公式法"。在此次实验中，分别利用面向过程与面向对象的思路去求解一元二次型方程的实数根。

**求解步骤：**

公式法求解一元二次型方程主要分为三步：

1. 判断是否为一元二次方程（使用方程的三个系数进行判断）
2. 判断是否有实数根
3. 公式法求解实根

**面向过程的实现**

编写 3 个函数，求解一元二次型方程的实根，若不满足一元二次方程的条件或无法求出实根，则输出原因。

**面向对象的实现**

编写一个一元二次型方程的类，并将上述三个功能作为成员函数，求解一元二次型方程的实根，若不满足一元二次方程的条件或无法求出实根，则输出原因。

#### 2. 点与圆的位置关系

点与圆的位置关系有三种：点在圆内、点在圆上、点在圆外。请编写两个类和一个全局函数，实现这三种关系的判断。

> **注意：** 圆类的成员中包含了圆心这个点类，属于类的组合。注意圆类构造函数的写法（初始化列表）。

**Point 类**

```
Point
────────────────────────────
-x:double
-y:double
────────────────────────────
+Point(double x, double y)  // 构造函数
+getX():double              // 获取 x 值
+getY():double              // 获取 y 值
+setX(double x):void        // 设置 x 值
+setY(double y):void        // 设置 y 值
```

**Circle 类**

```
Circle
────────────────────────────
-r:double
-center:Point
────────────────────────────
+Circle(double r, Point pt)  // 构造函数（初始化列表）
+getR():double               // 获取半径 r
+getCenter():Point           // 获取圆心
+setR(double r):void         // 设置半径
+setCenter(Point pt):void    // 设置圆心
```

**全局函数**

```cpp
int isPointInCircle(Point &pt, Circle &circle);
// 返回 -1：表示点在圆外
// 返回  0：表示点在圆上
// 返回  1：表示点在圆内
```

**主函数要求**

在主函数中：
- 创建圆的对象
- 创建点的对象
- 进行 3 种位置关系判断
- 输出判断结果
